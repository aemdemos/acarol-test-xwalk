This block is used to highlight a quotation or a specific passage within your content, often serving as a pull quote or a notable statement.

Implementation
This block’s table typically has 1 column and 2 rows, with the first row containing only the block name and optional variants.

  • The second row contains the quote itself

When rendered, CSS and JavaScript will style this text in a visually distinct manner, drawing attention to the quote or passage.

When to use

  • When you need to emphasize a brief excerpt of text or a key statement.
  • To showcase testimonials, opinions, or notable quotes within a larger document.
  • If you want to visually break up longer content with highlighted passages.

When to consider another block

  • If you need more complex formatting or layout elements than a simple quoted text.
  • When you’re embedding third-party reviews or outside sources that require additional metadata.
  • If your design calls for a more general text block rather than a highlighted quote.
