name
Table
description

This block is used to organize tabular data into a grid of rows and columns, making it easier for readers to compare and analyze information.

Implementation
This block’s table typically has multiple columns and multiple rows, with the first row containing only the block name and optional variants.

  • Each subsequent row represents a row of data.
  • Cells across the columns hold individual data points, headers, or labels as needed.

When rendered, CSS and JavaScript will maintain a clear, tabular format, allowing for organized display and easy scanning of the information.

When to use

  • When presenting structured data that benefits from a clear row-and-column layout.
  • To compare multiple data points side by side.
  • If a quick, at-a-glance view of related information is necessary.

When to consider another block

  • If your content consists of simple text or a single piece of information without comparisons.
  • When a more visual or narrative presentation (e.g., charts or cards) better communicates the data.
  • If interactivity or dynamic filtering is required beyond a static table.
