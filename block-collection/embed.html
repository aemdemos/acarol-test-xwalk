name
Embed (video)
description

This block is used to embed external video content (e.g., YouTube, Vimeo) directly into your page, providing a simple way to display interactive or third-party media.

Implementation
This block’s table typically has 1 column and 2 rows, with the first row containing only the block name and optional variants.

  • The following row should contain a single cell with a URL to the external content you want to display.
  • Optionally, if the provided URL is for a video, an image can also be provided in the same table cell. This image will serve as the poster for the video.
    • The image should be placed above the link in the cell.

When rendered, the CSS and JavaScript will transform each row into an embedded media component, allowing users to interact with or view the external content.

When to use

  • When you want to incorporate external videos directly on the page.
  • To enrich your content with dynamic, third-party media.
  • If providing visual or interactive content from external sources enhances the user experience.

When to consider another block

  • If your content is purely text or image-based and doesn’t require embedding.
  • When you need more design control or custom styling of third-party media.
  • If you want to avoid external scripts or slow loading times associated with embeds.
